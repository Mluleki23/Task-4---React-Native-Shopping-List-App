# 🛒 ShopList

A simple and user-friendly shopping list mobile app built with Expo and React Native.  
This app helps users create, manage, and track their shopping items بسهولة.

---

## 🚀 Features
- ➕ Add shopping items  
- 📋 View shopping list  
- ✅ Mark items as completed  
- 🗑️ Delete items  
- 📱 Clean and responsive UI  

---

## 🛠️ Technologies Used
- React Native  
- Expo  
- Expo Router  
- JavaScript  

---

ShopList/

---

## ▶️ Getting Started

### 1. Install dependencies
npm install

### 2. Start the app
npx expo start

### 3. Run the app
- Press **a** for Android  
- Press **i** for iOS  
- Scan QR code for mobile  

---

## ⚙️ Configuration
- App Name: ShopList  
- Orientation: Portrait  
- Platforms: Android, iOS, Web  
- Splash screen configured  
- Adaptive icons enabled
